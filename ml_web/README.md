# Machine Learning Web Application

A Streamlit-based web application for machine learning with a pastel color theme and modular architecture.

## Features

- **Data Upload**: Upload your own CSV file or use the provided Heart.csv dataset
- **Data Validation**: Perform exploratory data analysis (EDA) with visualizations
- **Data Preprocessing**: Handle missing values, scale features, and encode categorical variables
- **Model Training**: Train classification or regression models and evaluate their performance
